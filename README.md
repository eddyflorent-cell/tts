# TTS Enterprise KB

Troubleshooting Knowledge Base with Enterprise-grade mock error generator.

## Features

- Dynamic incident library
- Filters & search
- Image embedding (base64)
- Enterprise mock screenshot generator:
  - Windows 11 dialogs
  - macOS alerts
  - iOS / Android popups
  - Azure / Entra
  - Exchange / AD
  - VPN / Browser errors

## Deployment

Hosted via GitHub Pages.
